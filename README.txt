BANK QUEUE SIMULATION - README
==============================

How to Run the Simulation
-------------------------
1. Clone the Repository:
   git clone https://github.com/PramodHashantha/bank-queue-simulation.git
   cd bank-queue-simulation

2. Install Dependencies:
   pip install simpy

3. Run the Simulation:
   python bank_queue_simulation.py

4. Example Manual Input:
   Enter number of tellers: 3
   Enter mean interarrival time (minutes): 1
   Enter mean service time (minutes): 2
   Enter total simulation time (minutes): 480
